Sports Shop - Django Project

A complete e-commerce platform for sports equipment and apparel

Project Overview

Sports Shop is a full-featured e-commerce web application built with Django. It allows users to browse sports products, manage shopping carts, place orders, and leave reviews. The platform includes an admin interface for managing products, categories, orders, and user accounts.

Features

  • User Authentication: Registration, login, logout with secure password handling
  • User Profiles: Extended user profiles with address, phone, and bio
  • Product Catalog: Browse products with filtering and search functionality
  • Shopping Cart: Add/remove items, update quantities
  • Orders: Complete order management with status tracking
  • Reviews: Product reviews with ratings and moderation
  • Django Admin: Complete admin interface with custom filters and search
  • REST API: Full API with pagination and filtering (Django REST Framework)
  • Logging: Comprehensive logging of user actions and events
  • Signals: Django signals for automatic operations
  • Environment Configuration: Secure .env configuration

Technology Stack

  • Backend: Django 6.0.2
  • Database: SQLite (development), configurable for production
  • REST API: Django REST Framework
  • Frontend: Bootstrap 5 for responsive design
  • Configuration: python-dotenv for environment variables

Installation & Setup

Prerequisites

  • Python 3.8+
  • pip (Python package manager)
  • Virtual environment

Steps

  1. Clone the repository or extract the project files
  2. Create and activate virtual environment: 
    python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Linux/Mac
  3. Install dependencies: 
    pip install -r requirements.txt
  4. Copy .env.example to .env and configure: 
    cp .env.example .env
  5. Run migrations: 
    python manage.py migrate
  6. Create superuser for admin: 
    python manage.py createsuperuser
  7. Run development server: 
    python manage.py runserver
  8. Access the application:
    • Main site: http://127.0.0.1:8000/
    • Admin: http://127.0.0.1:8000/admin/
    • API: http://127.0.0.1:8000/api/v1/

API Endpoints

  • Products: GET /api/v1/products/ - List products with filters
  • Categories: GET /api/v1/categories/ - List categories
  • Reviews: GET/POST /api/v1/reviews/ - Product reviews
  • Orders: GET /api/v1/orders/ - User orders (authenticated)
  • Cart: GET/POST /api/v1/cart/ - Shopping cart operations
  • User: GET /api/v1/users/me/ - Current user profile

Models

  • UserProfile: Extended user information with addresses and preferences
  • Category: Product categories with slugs
  • Product: Product details with pricing, stock, and images
  • ProductImage: Additional product images
  • Order: Customer orders with shipping and payment info
  • OrderItem: Items within an order
  • Cart: Shopping cart per user
  • CartItem: Items in shopping cart
  • Review: Product reviews with ratings
